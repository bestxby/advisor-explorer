## 🌟 核心用户体验 (User-Centric Features)

### 🌀 1. 沉浸式 3D 仿生粒子空间 (WebGL Interactive Galaxy)
* **形态即心智**：全站背景由 **110,000 个粒子** 构成的三维星空构成。当您回答测评问题或在不同学术方向间切换时，粒子星空会触发平滑而震撼的“爆炸性形态重组”——在**双扭线（Infinity）**、**量子网格**与**动态波浪**等多种形态间切换。
* **变色龙主题同步**：当您匹配到不同的学派时，全站会动态泛起对应学派的霓虹微光（量子青、极客紫、极光绿），让学术流派的转换充满仪式感。

### 🧠 2. 多维度“学术人格”深度匹配 (Academic Style Matchmaker)
* **摆脱单一标准**：不仅对比研究方向，更从**代码偏好**、**导师管理风格**、**论文发表节奏**等多个软硬实力维度进行交叉算力建模。
* **$O(1)$ 高速精准推荐**：基于哈希倒排索引计算，当您完成问卷时，平台将在 0.1 毫秒内为您输出契合度最高的研究学派、导师排行，以及详细的“科研优势（Strengths）”与“避坑指南（Cautions）”。

### 📊 3. 学术市场跨界横向大对比 (Interactive School Comparison)
* 平台集成了可视化三维数据横向对比面板，让您在做决定前，一眼洞悉不同方向的现实与未来：
  * 📈 **行业市场空间**（Job Market Heat & Salary Ceiling）
  * 🎛️ **学派上手难度**（Maturity & Tapeout Dependency）
  * ⚡ **35岁风险预警与护城河评估**（Risk Mitigation & Moat）

### 🗺️ 4. 千人千面的“三阶段学术路线图” (Personalized Research Roadmap)
* 系统会自动为您量身定制一份结构清晰的**成长指南**：
  * **第一阶段：心智重构与极速起步**（核心突破：文献阅读与微架构破局）
  * **第二阶段：硬核重器与学术输出**（核心突破：完整流片体系或系统重构）
  * **第三阶段：学术毕业与职业收割**（核心突破：高水平会议（ISCA/ASPLOS）突围）
  * **里程碑体系**：提供高、中、低优先级分层任务卡片，支持悬浮展开。

### 🎨 5. 2K 视网膜级海报导出与学术报告 (Retina Export Service)
* 🚀 **超高清手绘 Canvas 社交海报**：一键生成 Retina 2K 级别的“学术契合度认证海报”，内置学术标签与契合度雷达，方便在社交网络分享。
* 📝 **无缝 PDF 打印与纯 Markdown 报告**：生成排版精美、支持多端自适应阅读的专属科研探索报告，随时备查与打印。

## 🛠️ 启动与极速体验说明 (For Developers & Power Users)

如果您是极客用户，希望在本地运行或自定义导师数据库：

### 1. 启动全套系统
```bash
# 1. 安装精简依赖
npm install

# 2. 启动本地高帧率开发服务器
npm run dev
```

### 2. 录入您自己的导师数据库
您可以随时修改 `src/data/professors.json` 和 `src/data/directions.json`。修改完成后，运行以下一键校验命令：
```bash
npm run validate-data
```
终端会以高亮彩虹色输出数据的一致性检验报告，拦截任何格式或 ID 漏写错误，保障您的数据库固若金汤！
